# Lecture Intelligence System

Minimal scaffold for the Lecture Intelligence System (backend + simple UI).

Quick start

1. Create a Python virtual environment and activate it.
2. Install dependencies: pip install -r requirements.txt
3. Run the API: uvicorn app.main:app --reload
4. Open the Chainlit UI (if configured) or call the /chat API.

Project layout

See the repository tree for a suggested layout. Data, models and venv are gitignored by default.
